## Run Locally

Clone the project

```bash
  git clone https://github.com/boneyrox/spotifyLite
```

Go to the project directory

```bash
  cd spotifyLite
```

Install dependencies

```bash
  npm install
```
or
```bash
  yarn install
```

Start the server

```bash
  npm run start
```
or

 ```bash
  yarn run start
```


## Tech Stack

**Client:** React, TailwindCSS

**Dependencies:** js-cookie, react-icons, tiny-slider-react, axios, react-spotify-auth
